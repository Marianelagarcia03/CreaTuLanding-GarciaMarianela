# Proyecto final React

**Descripción**

Este repositorio fue creado como proyecto final del curso de React js de Coderhouse.
Se trata de una tienda e-commerce de bebidas en la que se aplicaron todos los conceptos aprendidos durante el cursado.

## Cómo usar la aplicación
**Saludo inicial en Home**  
Al ingresar al sitio, verás un mensaje de bienvenida en la página principal.
**Explorar productos por categorías**  
Puedes navegar por las diferentes categorías de productos.
**Ver detalle de un producto**  
Al hacer clic en un producto, se abrirá una página con información detallada, incluyendo imagen, descripción y precio.
**Agregar productos al carrito**  
Desde la página de detalle del producto, podrás seleccionar la cantidad y agregarlo al carrito de compras.
**Administrar el carrito**  
En el carrito, podrás:
   - Ver todos los productos agregados.
   - Eliminar productos.
**Generar una orden de compra**  
Al finalizar, podrás completar un formulario con tus datos para generar una orden.
